What are the best practices associated with using classes vs. ids?

The best practices between using classes vs ids is if you want to work with multiple elements While for ID it best for using on single elements. Class help you work with multiple aspects of your web sites while Id is best used on one single part of you web page. ID also able to be use in your web page to jump between parts.
