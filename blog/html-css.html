What are the best practices associated with using classes vs. ids?

Classes and ID are both use as selectors in html and css to style your web page in simple terms You telling what your should look like.

Class can be use to affect multiple elements and use multiple across the whole web site. ID can only be use on one single element at time but it can be use to navigate between different points in your web page.
